# Context Directory

Domain knowledge shared across all skills.

- `my-business.md` — Your business details: what you do, who you serve, your offers, how you generate leads. Filled by the setup wizard.
- `my-voice.md` — Your communication style: tone, phrases, samples. Filled by the setup wizard.

Add more context files as needed (e.g., `my-icp.md` for ideal customer profiles, `case-studies.md` for client examples).

Skills reference these files to produce accurate, personalized output.
